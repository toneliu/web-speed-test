# Web Speed Test - 网络测速工具

一个简单实用的网页网络测速工具，帮助用户快速测试网络带宽和性能。

## 功能特点

- 🌐 **网速测试** - 实时测量下载和上传速度
- 📊 **详细数据** - 显示延迟、抖动等网络指标
- 📱 **响应式设计** - 支持桌面端和移动端
- ⚡ **快速准确** - 采用先进的测速算法
- 🎨 **简洁界面** - 直观的用户体验

## 技术栈

- 前端: HTML5 + CSS3 + JavaScript
- 测速引擎: JavaScript + Web APIs
- 部署: GitHub Pages

## 快速开始

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/toneliu/web-speed-test.git
cd web-speed-test
```

2. 直接在浏览器中打开 `index.html` 文件

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

3. 访问 http://localhost:8000

## 使用方法

1. 点击"开始测试"按钮
2. 等待测速完成
3. 查看详细的测速结果

## 测速指标说明

- **下载速度**: 网络下载数据的速度（Mbps）
- **上传速度**: 网络上传数据的速度（Mbps）
- **延迟(Ping)**: 数据包往返时间（ms）
- **抖动**: 网络延迟的波动程度（ms）
